# codex-feishu v0.4.1

Restores immediate acknowledgement delivery and updates the acknowledgement text.

## Highlights

- `cc-connect-ack` now retries briefly when the `message.received` hook fires before the target session is ready for `cc-connect send`.
- The immediate acknowledgement text is now `收到正在输出，请等等我。`.
- Windows and Linux ack hooks use the same text and retry behavior.

## Install

Normal Windows and Linux installation remains unchanged.

Windows hook is generated by:

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\install.ps1
```

Linux hook is generated by:

```bash
bash ./scripts/install-linux.sh
```

## Verify

```powershell
powershell.exe -NoProfile -ExecutionPolicy Bypass -File .\scripts\test.ps1
```

Expected:

- generated ack hook contains the new acknowledgement text;
- PowerShell parser, secret scan, and install smoke checks pass.
